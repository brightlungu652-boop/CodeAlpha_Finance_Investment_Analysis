# CodeAlpha_Finance_Investment_Analysis
CodeAlpha Finance &amp; Investment Analysis Internship projects implemented in Python, including stock analysis, cryptocurrency portfolio analysis, and investment portfolio creation with quantitative risk analysis and visualizations.
# CodeAlpha Finance & Investment Analysis Internship

## Overview

This repository contains the projects I completed during my CodeAlpha Finance & Investment Analysis Internship. The work focuses on analyzing financial data using Python and applying basic investment and risk management concepts.

The project is divided into three tasks:

- Stock Analysis Report
- Cryptocurrency Portfolio Analysis
- Investment Portfolio Creation

---

## Task 1 – Stock Analysis

For this task, I selected three US companies:

- Microsoft (MSFT)
- Apple (AAPL)
- NVIDIA (NVDA)

The analysis includes:

- Historical price trends
- Market capitalization
- P/E ratio
- Performance metrics
- Recent company news
- Summary report with charts

Files:
- `Task1_Stock_Analysis.ipynb`
- `Stock_Analysis_Report.pdf`
- `stocks_summary.xlsx`

---

## Task 2 – Cryptocurrency Portfolio Analysis

The cryptocurrencies analyzed are:

- Bitcoin (BTC)
- Ethereum (ETH)
- Solana (SOL)

The project includes:

- Historical price analysis
- Daily returns
- Annual return
- Annual volatility
- Sharpe Ratio
- Sortino Ratio
- Maximum Drawdown
- Value at Risk (95%)
- Correlation analysis
- Data visualizations

Files:
- `Task2_Cryptocurrency_Analysis.ipynb`
- `Cryptocurrency_Portfolio_Report.pdf`
- `crypto_summary.xlsx`

---

## Task 3 – Investment Portfolio Creation

A hypothetical investment portfolio worth ₹100,000 was created using:

### Stocks
- Microsoft
- Apple
- NVIDIA

### ETFs
- Vanguard S&P 500 ETF (VOO)
- Invesco QQQ ETF (QQQ)

### Cryptocurrencies
- Bitcoin
- Ethereum
- Solana

The report explains the asset allocation and the reasons for including each investment.

Files:
- `Task3_Portfolio.ipynb`
- `Investment_Portfolio_Report.pdf`
- `portfolio.xlsx`
- `portfolio_summary.xlsx`

2. Install the required packages:

```bash

```

3. Open the Jupyter notebooks and run the cells.
